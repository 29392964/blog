Download Mgweb v1.1

Features

Cross-platform

Mgweb runs on OSX, Linux and Windows operating systems without a hustle. Binaries are cross-complied with Go and available for 32/64 bit systems. You can even run it on RaspberryPi

Easy to Install

Mgweb comes as a single binary file that's ready to go. You can install it manually, via Docker or using Homebrew on OSX which is updated regularly and is super convenient.

Zero Dependencies

No need to install anything on your machines or services. Just download and run. To get started Mgweb just needs a browser and a Mongodb server to connect to.

Simple and Clean

Mgweb was designed to be very simple and clean UI to browse or edit collections.
